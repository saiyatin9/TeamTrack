Connect to Firebase

TeamTrack uses your own private Firebase project for real-time sync across all employee devices. It's completely free for small teams (Spark plan).

  1. Go to console.firebase.google.com → Create a project
  2. In your project: Build → Firestore Database → Create database (choose Test mode)
  3. Go to Project Settings ⚙ → Your apps → Add web app (</>)
  4. After registering the app, copy the firebaseConfig values below

Your data lives in your own Firebase project. TeamTrack has zero access to it.
Default admin password: admin123 (change it after login)
